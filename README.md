# BigQuery Release Insights Event Talks App

A lightweight web application that aggregates and presents Google BigQuery release notes and event insights in an easy-to-read interface.

## Features

* Fetches BigQuery release updates
* Parses release notes from official sources
* Displays updates in a clean web interface
* Lightweight Flask-based architecture
* Simple HTML, CSS, and JavaScript frontend

## Project Structure

```text
bigquery-release-insights-event-talks-app/
├── app.py

│   └── index.html
├── static/
│   ├── css/
│   │   └── style.css
│   └── js/

```

## Installation

Clone the repository:

```bash
git clone https://github.com/VictorUIT/AI-with-antigravity.git
cd AI-with-antigravity
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
python app.py
```

Then open your browser and navigate to:

```text
http://localhost:5000
```
